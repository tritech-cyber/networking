Subnetting Example
The first entry in the Class A table (/10 subnet mask) borrows two bits 
(the leftmost bits) from the host portion of the network for subnetting, 
then with two bits you have four (22) combinations, 00, 01, 10, and 11. 
Each of these will represent a subnet.

Binary Notation                                        Decimal Notation 
--------------------------------------------------     ----------------- 
xxxx xxxx. 0000 0000.0000 0000.0000 0000/10    ------> X.0.0.0/10 
xxxx xxxx. 0100 0000.0000 0000.0000 0000/10    ------> X.64.0.0/10 
xxxx xxxx. 1000 0000.0000 0000.0000 0000/10    ------> X.128.0.0/10 
xxxx xxxx. 1100 0000.0000 0000.0000 0000/10    ------> X.192.0.0/10
